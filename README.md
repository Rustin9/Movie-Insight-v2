#  AI Movie Insights

An AI-powered movie briefing app. Enter a movie title and get an instant plot hook, thematic analysis, and trivia facts — generated by GPT-4o-mini from a curated TMDB dataset.


---

## What It Does

- **Plot Hook** — A one-sentence AI-generated summary designed to hook you in
- **Core Themes** — Deeper thematic analysis beyond genre labels
- **Trivia & Facts** — Production history, legacy, and behind-the-scenes facts
- **Movie Poster + Metadata** — Rating, genres, director, cast from TMDB
- **Dark / Light Mode** — Persisted across sessions

---

## Architecture

This system is **retrieval-first**. Before calling the LLM, the app retrieves a structured movie record from a curated JSON dataset (stored in Google Cloud Storage). That record — title, overview, cast, director, genres, rating — is injected directly into the GPT-4o-mini prompt as context.

```
TMDB API → ETL (GitHub Actions) → GCS Bucket → Next.js API → GPT-4o-mini → UI
             Bronze → Silver → Gold
```

The ETL runs **every 12 hours** via a scheduled GitHub Actions workflow, keeping the catalog fresh automatically.

---

## Data Pipeline

| Stage | File | Description |
|---|---|---|
| Bronze | `data/raw/bronze/movies_raw.json` | Raw TMDB API response |
| Silver | `data/raw/silver/movies_enriched.json` | Enriched with cast, director, poster |
| Gold | `data/raw/gold/movies.json` | Final curated dataset used by the app |
| Meta | `data/raw/gold/etl_meta.json` | Run timestamp, movie count, elapsed time |

The ETL workflow commits the gold file back to the repo on every run, so the app always has a local fallback even when GCS is unavailable.

## Supported Movies

The catalog is refreshed from TMDB's popular movies endpoint every 12 hours (~60 titles). Examples:

- Project Hail Mary
- Avatar: Fire and Ash
- Scream 7
- Zootopia 2
- The Super Mario Bros. Movie

Partial title matching is supported — typing `"Hail Mary"` resolves to `"Project Hail Mary"`.

**Out of scope:** Real-time lookups, recommendations, watch lists, movies not in the catalog.
